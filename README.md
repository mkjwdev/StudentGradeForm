# StudentGradeForm
An Excel VBA student grade form made in 2016.

### Original Work

Copyright (c) 2016 Martino Kang

## Running locally

1. Download ZIP
2. Extract ZIP
3. Open Excel
4. Go to Developer Tab
5. Click on Visual Basic
6. Right click select Import File
7. Select StudentGradeForm.frm
8. Have fun
